# Ethiopian Soldier vs Italian Soldier - Java 2D Game

A 2D shooting game made as a class project, inspired by the historic **Battle of Adwa (1896)** — and named it axume after my favorite ethiopian artifact.  


---

## 🎮 Gameplay

- Control the **Ethiopian soldier** and defend against Italian soldiers.  
- Enemies spawn from **both directions**, but **only you can shoot**.  
- survive and test your reflexes.

---

## ⚡ Features

- Simple 2D top-down mechanics in Java Swing.  
- Enemy soldiers spawn dynamically from left and right.  
- Score tracking and no sound effects yet.

## 🛠 Tech

- Java  
- Java Swing
